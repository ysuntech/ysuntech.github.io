    使用hugo搭建博客系统

    介绍

    作为web技术从业人员，经常有一些笔记心得需要记录。可以选择在csdn或者oschina等站点写博客。也有人购买VPS，用wordpress搭建自己的博客。这样比较自由，但购买VPS，搭建lamp环境需要一定的Time和Money。其实有更简单的解决方案：使用hugo和github pages搭建全静态免费博客。

    搭建博客

    让我们开始吧,按步骤来：

    1. 下载hugo软件，图省事可以直接下载二进制版本

    2. 将下载得到的hugo二进制文件加入PATH环境变量。这样我们可以从命令行输入”hugo”随时调用hugo

    3. 初始化博客项目

      cd ~
# 创建名为hugo_blog的博客站点
hugo new site hugo_blog
